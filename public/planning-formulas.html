1 Width Hierarchy & Constraints

Golden Rule

Three widths must always satisfy this strict ordering:

Film Width  ≤  Sleeve Width  <  Roll Width

Roll Width

The machine's maximum web width. Film must fit within this. Sleeve must also be narrower than the roll.

rollWidth = machine.maxWebWidth

Sleeve Width

Must be wide enough to hold the film + trimming. Must be strictly less than roll width.

sleeve < rollWidth
film ≤ sleeve

Film Width

Actual film roll used. Defined by printing + trimming. Must not exceed sleeve width.

filmWidth = printing + 2 × trim

3 Lane Width & Printing Width

// Lane Width = width each UPS lane occupies on the sleeve
laneWidth = actualWidth + trimmingSize

// Printing Width = total width of all lanes combined
printingWidth = acUps × laneWidth

// Max AC UPS a given sleeve can accommodate
maxAcUps = floor( sleeveWidth / laneWidth )

Example

Actual Width = 250mm, Trimming = 10mm
laneWidth = 260mm
For Sleeve 1130mm → maxUPS = floor(1130/260) = 4

Note

Plans are generated by iterating sleeves first, then computing how many UPS fit in each sleeve. Not by iterating UPS first.

4 Film Width Formula

// Film Width = actual film roll width needed for this plan
filmWidth = printingWidth + 2 × trimmingSize

// VALIDATION — reject plan if film exceeds sleeve
IF filmWidth > sleeveWidth → REJECT PLAN

// Guaranteed constraint after validation:
filmWidth ≤ sleeveWidth < rollWidth

Valid (Film = Sleeve)

printing=1120, trim=10, sleeve=1140
filmWidth = 1120+20 = 1140 ≤ 1140 ✓

Invalid (Film > Sleeve)

printing=1125, trim=10, sleeve=1130
filmWidth = 1125+20 = 1145 > 1130 ✗ → REJECTED

5 Sleeve Selection & Validation

// All available sleeves from Tool Inventory (status = "Available")
SLEEVE_TOOLS = tools.filter( toolType == "Sleeve" && status == "Available" )
               .sort( ascending by printWidth )

// Sleeve is valid for a UPS if it can physically accommodate the lanes
VALID IF: sleeveWidth ≤ rollWidth  &&  floor(sleeveWidth / laneWidth) > 0

// After film calculation, reject if film > sleeve
REJECT IF: ( printingWidth + 2 × trim ) > sleeveWidth

6 Cylinder Selection (Case 1 & Case 2)

// Minimum cylinder requirement based on printing area
req = filmWidth + 100     // = printingWidth + 2×trim + 100 (50mm each side of film)

CASE 1 (Normal — sleeve is wider than req):
  IF req < sleeveWidth
  THEN minCylinder = req   // cylinder sized for printing area

CASE 2 (Tight — req meets or exceeds sleeve):
  IF req ≥ sleeveWidth
  THEN minCylinder = sleeveWidth + 100   // cylinder sized for sleeve + clearance

// Cylinder must be available in inventory (status = "Available")
validCylinders = CYLINDER_TOOLS.filter( printWidth ≥ minCylinder )

Case 1 — Example

printing=660, trim=10, sleeve=1000
filmWidth = 680
req = 780
780 < 1000 → CASE 1 → minCyl = 780mm

Case 2 — Example

printing=690, trim=10, sleeve=740
filmWidth = 710
req = 810
810 ≥ 740 → CASE 2 → minCyl = 740+100 = 840mm

One plan row is generated per (Sleeve × UPS × Cylinder) combination — multiple cylinders show different cost/size options.

7 Waste Calculations

// Side Waste — trimming margins on the film (both sides)
sideWaste = 2 × trimmingSize

// Dead Margin — sleeve width beyond the film edge
deadMargin = sleeveWidth - filmWidth   // always ≥ 0 (guaranteed by filmWidth ≤ sleeve)

// Total Waste — full unused material per UPS combination
totalWaste = sideWaste + deadMargin             = sleeveWidth - printingWidth

Film WidthprintingWidth + 2×trimActual film roll width consumed
Side Waste2 × trimmingSizeCut-off trimming margins on both sides of film
Dead MarginsleeveWidth − filmWidthSleeve area beyond the film edge (always ≥ 0)
Total WastesideWaste + deadMargin= sleeveWidth − printingWidth (minimum = best)

8 Plan Generation Flow

For each Machine — filter machines by department = "Printing". If specific machine selected, use only that machine. Get rollWidth = machine.maxWebWidth.

For each Available Sleeve — from Tool Inventory (status = "Available"), sorted by width ascending. Skip if sleeve ≥ rollWidth.

Calculate maxAcUps — floor(sleeveWidth / laneWidth). Skip sleeve if maxAcUps = 0.

For each UPS (1 to maxAcUps) — compute printingWidth = acUps × laneWidth, then filmWidth = printingWidth + 2×trim. Reject if filmWidth > sleeveWidth.

Select Cylinders — compute req = filmWidth + 100, apply CASE 1 or CASE 2 to get minCylinder. Filter available cylinders ≥ minCylinder.

Generate Plan Row — one row per (Machine × Sleeve × UPS × Cylinder). Calculate waste, cost, production metrics.

Sort & Mark BEST — sort by totalWaste → deadMargin → sideWaste → max acUps. First row = BEST.

10 Cylinder Cost Formula

// Cylinder dimensions
cylRepeatLength = cylinder.repeatLength  // from Tool Master (mm)
cylAreaSqMm = cylinderWidth × cylRepeatLength
cylAreaSqInch = cylAreaSqMm / 645.16                 // 1 sq.inch = 645.16 sq.mm

// Area-based cylinder cost (when rate is set)
cylCostByArea = cylAreaSqInch × ratePerSqInch × noOfColors

// Fallback: per-color cost (when rate = 0)
cylCost = rate > 0   ? cylCostByArea   : noOfColors × cylinderCostPerColor

11 Production Metrics

// Repeat UPS — how many impressions per cylinder revolution
baseCirc = ceil( jobHeight / 12 ) × 12   // rounded up to nearest 12mm
repeatUPS = floor( baseCirc / jobHeight )

// Total UPS = across UPS × repeat UPS
totalUPS = acUps × repeatUPS

// Required material (metres)
reqRMT = ceil( quantity / totalUPS )
totalRMT = ceil( reqRMT × 1.01 )   // +1% for startup/waste

// Machine time and cost
totalTime = totalRMT / ( speed × 60 )   // hours (speed in m/min)
planCost = totalTime × machine.costPerHour

// Grand total and unit price
grandTotal = planCost + processCost + cylCost
unitPrice = grandTotal / quantity
